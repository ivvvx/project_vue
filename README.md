# 前端项目集

## 项目列表

### 1. Vue2 智慧商城移动端应用

- 技术栈：Vue2 + Vuex + Vue Router + Vant
- 功能：商品浏览、购物车、订单流程、用户登录
- 在线演示：https://ivvvx.github.io/project_vue/vue2-smart-shopping/
- 本地运行：`cd vue2-smart-shopping && npm install && npm run serve`

### 2. Vue3 大事件后台管理系统

- 技术栈：Vue3 + Pinia + Element Plus + Vite
- 功能：文章管理、用户管理、权限控制
- 在线演示：https://ivvvx.github.io/project_vue/vue3-bigEvent-admin/
- 本地运行：`cd vue3-bigEvent-admin && npm install && npm run dev`

## 说明

由于后端服务暂未部署，线上演示版本已内置 Mock 数据，完整展示页面交互与功能。后端 API 设计与接口文档详见各项目目录。
